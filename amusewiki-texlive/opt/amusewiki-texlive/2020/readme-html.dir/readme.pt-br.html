Bem-vindo ao TeX Live

A distribuição TeX Live oferece um completo sistema TeX para uma variedade de plataformas GNU e Unix, Macintosh, Windows entre outras. Ela contém programas para editar, digitar, pré-visualizar e imprimir documentos TeX em diferentes idiomas, e uma grande coleção de macros TeX e bibliotecas de fontes.

A distribuição inclui uma documentação geral extensa sobre TeX, bem como a documentação para os pacotes inclusos.

Para informações sobre licença, favor veja a seção principal LICENSE.TL e LICENSE.CTAN destes arquivos.

Por onde começar

Recomendamos fortemente que você leia o guia TeX Live antes de efetuar a instalação. Ele inclui instruções passo-a-passo, uma descrição dos principais diretórios, além de outras informações.

Se você está instalando a partir de uma imagem em DVD, o guia em Inglês e todas suas traduções estão no subdiretório texlive-doc.

Após a instalação, versões em Inglês do Guia estão em texmf-dist/doc/texlive/, em ambos os formatos html e pdf. Traduções para outros idiomas também estão disponíveis em texmf-dist/doc/texlive.

Fornecemos também uma lista de links para as documentações disponíveis em PDF e HTML para os pacotes e programas inclusos.

Se você está instalando a partir da Internet, os links acima não funcionarão. Neste caso, você pode consultar a documentação TeX Live na Internet.

Para uma lista das mudanças mais importantes no lançamento deste ano, veja a seção Histórico de Lançamentos no final da documentação.

Instalação

O programa de instalação (script) é nomeado install-tl e está no nível superior da árvore de diretórios. Você pode invocá-lo com perlinstall-tl . No Windows, você pode executar o arquivo em lote anexado ( install-tl-windows.bat . Nos Macs, provavelmente é melhor instalá-lo via MacTeX , que possui seu próprio instalador nativo (também está no DVD, ao lado de texlive em que você está agora).

install-tl tem dois modos de operação:

-gui
(padrão no Windows e Macs) que executa um Instalação padrão simples, usando uma interface gráfica, mas que, no entanto, permite muitas opções avançadas, se necessário.
-no-gui
(padrão em todos os outros sistemas operacional), que fornece todas as opções disponíveis, usando uma interface de texto gráfica simplificada.

Atualizações de pacotes pós-instalação, assim como outras configurações são realizadas com o programa (script) tlmgr (TeX Live Package Manager), que também opera em ambos modos: texto e GUI.

Você pode executar a opção --help na linha comando para ambos: install-tl e tlmgr que fornece Informações mais detalhadas sobre esses programas. Veja os links da documentação acima para (muito) mais informações.

Links

Finalmente, algumas páginas web relacionadas, para consulta. Estas páginas não estão incluídas no TeX Live:

  • http://tug.org/begin.html - se você é novato no TeX e/ou deseja material introdutório.
  • http://tug.org/interest.html - lista geral sobre TeX, incluindo documentação, sistemas, pacotes, fontes, e mais.
  • http://tug.org/ctan.html - uma porta de entrada para o Comprehensive TeX Archive Network (CTAN), que é um espelho online de todas as coisas do TeX.
  • http://tug.org/usergroups.html - lista de grupos de usuários TeX ao redor do mundo, com informações de contato. Recomendamos que você faça parte do grupo de sua escolha, para dar suporte ao TeX Live e outras atividades TeX.
